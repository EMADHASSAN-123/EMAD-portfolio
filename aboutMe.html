ما قبل الرحلة

قبل دخولي التعليم الجامعي، كانت لدي تساؤلات اللهمتني لتخصصي : ماذا لو كان لدى شخص فكرة ويريد إيصالها للعالم؟ أو لديه محتوى مفيد يود تقديمة؟ أو حتى مثشروع محلي يسعى للتوسع الجغرافي لكنه يواجه عوائق مالية؟ كانت هذه الأسئلة تدور في ذهني حتى وجدت الحل الساطع الذي أضاء طريقي وهو عبر تقنية المعلومات

تحديد رحلتي

اخترت تخصص تقنية المعلومات، وكان شغفي بها يدفعني لاكتشاف كيف يمكن لهذه لتقنية المعلومات أن تحدث فرقا. وبعد عام من الدراسة، أدركت أنني أستطيع تحقيق أهدافي بطرق متعددة، حيث يمتلك كل مجال برمجياته ولغاته الخاصة.

بداية رحلتي

بدأت رحلتي مع تقنيات الويب بتعلم HTML لهيكلة صفحات الويب، و CSS لتنسيقها واظهار جمالها،ثم قمت بإضافة عنصر التفاعل عبر javascript مما جعلني أشعر بأني أخلق تجربة فريدة للمستخدمين لم يتوقف الأمر عند هذا الحد، تعلمت أيضا كيف أجعل المواقع تقدم محتوى حديثا وتفاعليا باستخدام PHP و استعلامات SQL

مهارات وتقنيات الرحلة

خلال رحلتي التعليمية، اكتسبت العديد من المهارات الشخصية فبعض منها مكتسبة من خلال كورسات تعليمية وبعضها من خلال التعاملات اليومية والانشطة الجماعية مع الاخرين.
تعلمت كيفية البحث والتحليل، مما ساعدني على فهم المواد بشكل أعمق. بالإضافة إلى ذلك، طورت مهارات التواصل والعمل الجماعي، حيث كانت المشاريع الجماعية جزءًا أساسيًا من تجربتي. كما تعلمت كيفية إدارة الوقت بفعالية، مما مكنني من تحقيق التوازن بين الدراسة الجامعية والتطوير الذاتي.

    و أبرز المهارات البرمجية
  • تصميم وتطوير المواقع الثابتة باستخدام HTML, CSS, JS
  • تطوير المواقع التفاعلية
  • تطوير المواقع الديناميكية وربطها بقواعد البيانات
  • تطوير المواقع الالكترونية باستخدام نظام ادارة المحتوى WordPress
  • تقنية Ajax لتحديث الموقع بشكل غير متزامن وادارة الطلبات
  • استخدام تقنية Ajax مع مكتبة jQuery لتقليل الكود وزيادة اداء الموقع
  • تحسين المظهر العام للموقع حسب طلب دراسة تجربة المستخدم
  • ربط الموقع بواحهة تطبيقات المستخدم API حسب تكامل الموقع
  • WCAG تحسين قابلية الوصول للموقع لذو الاحتياجات الخاصة
